# Industrial Webpage

A responsive industrial website developed using React and React Router, featuring seamless navigation between multiple pages, modern UI design, service showcases, company information, and contact sections.

## Features

* Multi-page navigation using React Router
* Responsive design
* Modern user interface
* Service showcase section
* Company information page
* Contact page

## Technologies Used

* React.js
* React Router
* JavaScript
* HTML5
* CSS3
* Vite
